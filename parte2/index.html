    Bootstrap

    Bootstrap es un framework de código abierto para el desarrollo de sitios web y aplicaciones web. Fue creado por Mark Otto y Jacob Thornton en Twitter y lanzado en 2011. Bootstrap facilita la creación de interfaces de usuario modernas y responsivas gracias a su colección de herramientas y componentes predefinidos.

    Popover

    Un popover es un componente de interfaz de usuario que muestra contenido emergente o flotante, generalmente utilizado para proporcionar información adicional o contextos específicos en respuesta a una acción del usuario, como pasar el ratón sobre un elemento o hacer clic en un botón.


    Tooltip

    Un tooltip es un componente de interfaz de usuario que muestra un pequeño cuadro de texto emergente cuando el usuario pasa el cursor sobre un elemento, como un botón, un enlace o una imagen.

    Slider

    Un slider, también conocido como carrusel o carousel, es un componente de interfaz de usuario que permite a los usuarios navegar a través de una serie de elementos de contenido, como imágenes, texto o vídeos, desplazándolos horizontalmente o verticalmente.

    Megamenú

    Un megamenú es un tipo de menú desplegable que se utiliza en sitios web para mostrar un gran número de opciones de navegación en una única área de menú. A diferencia de los menús desplegables tradicionales.

    Barras de Progreso

    Las barras de progreso son componentes visuales que muestran el estado de una tarea en curso mediante una barra que se llena progresivamente. Son útiles para indicar el avance de descargas, cargas, procesamiento de datos, o cualquier otra tarea que tenga una duración predecible.
